# ClipForge AI

AI-powered SaaS to turn long-form videos into viral short clips.

## Features
- Stripe subscriptions
- YouTube upload detection
- AI clipping
- Auto captions
- Instagram & TikTok posting

## Stack
- Next.js
- Node.js
- Python (Whisper)
- FFmpeg
